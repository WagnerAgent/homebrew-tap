# WagnerAgent Homebrew Tap

This is the official Homebrew tap for [Wagterm](https://github.com/WagnerAgent/wagterm).

## Installation

```bash
brew tap WagnerAgent/homebrew-tap
brew install --cask wagterm
```

## Available Casks

| Cask | Description |
|------|-------------|
| wagterm | Open-source AI SSH desktop client |

## Updating

```bash
brew update
brew upgrade --cask wagterm
```

## Uninstalling

```bash
brew uninstall --cask wagterm
brew untap WagnerAgent/homebrew-tap
```
